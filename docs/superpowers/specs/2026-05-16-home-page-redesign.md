# Homeページリニューアル設計

## 概要

現在のHomeページとAboutページを統合し、シングルページ構成のHomeページにリニューアルする。
ナビゲーションを **Home / Blog / Projects** の3ページ構成に変更する。

## 目的

個人の名刺代わりとして、訪問者がスクロールだけで「誰・何ができる・何を作った・何を書いた・どう連絡する」を把握できるようにする。

## 現状

- Home (`/`): 汎用的なウェルカムメッセージのみ
- About (`/about`): プレースホルダー（「Write your introduction here.」）
- ナビゲーション: Home / About / Blog / Projects の4ページ

## 設計

### ページ構成の変更

- `/about` ページを廃止
- ヘッダーナビゲーションからAboutを削除
- ナビゲーション: **Home / Blog / Projects** の3ページ構成

### Homeページのセクション構成

上から順に以下の5セクションを配置する。

#### 1. ヒーローセクション

- 名前: アサオカ タクヤ
- 肩書き: Software Engineer / SRE
- 一言紹介: 短いキャッチコピー（内容は実装時に決定）

#### 2. スキルセクション

- 使用技術をタグまたはアイコンで一覧表示
- カテゴリ分けや経歴は含めない
- 具体的な技術スタックの内容は実装時に決定

#### 3. Projectsピックアップセクション

- 最新3件のプロジェクトをカード形式で表示
- 既存の `ProjectCard` コンポーネントを活用
- 「もっと見る」リンクで `/projects` 一覧ページへ遷移

#### 4. Blog最新記事セクション

- note.comからRSSで取得した最新3件を表示
- 既存のRSS取得ロジック（`/blog` ページ）を再利用
- 「もっと見る」リンクで `/blog` ページへ遷移

#### 5. Contactセクション

- GitHub: 既存の `consts.ts` のリンクを使用
- メールアドレス: 新規追加（`consts.ts` に追加）

### 影響範囲

| ファイル | 変更内容 |
|---------|---------|
| `src/pages/index.astro` | セクション構成を全面リニューアル |
| `src/pages/about.astro` | 削除 |
| `src/components/Header.astro` | ナビゲーションからAboutを削除 |
| `src/consts.ts` | メールアドレスの定数を追加 |
| `src/styles/global.css` | 各セクション用のスタイル追加 |

### スコープ外

- Projectsページの詳細設計（別途検討）
- Blogページの改修
- デザイン（色、フォント、アニメーション等）の詳細決定
